Bachelor Thesis

Warehouse System - Voice & Touch Interface

Project Overview

My bachelor thesis project involved the development of a comprehensive system for efficient product plucking in a warehouse setting. Our application incorporated a unique concept where employees could simultaneously utilize voice and touch interfaces for seamless plucking operations.

Technical Implementation

  • Frontend: SwiftUI interface supporting both iPhone and watchOS platforms
  • Backend: Robust Spring Boot API for backend operations
  • Speech Technology: Speech recognition and text-to-speech capabilities
  • Security: User authentication system
  • Management: Comprehensive warehouse management capabilities

Key Features

  • Dual voice and touch interface for hands-free operation
  • Cross-platform support (iPhone and Apple Watch)
  • Real-time warehouse inventory management
  • Voice-guided picking instructions
  • Seamless user authentication

Project Impact

The system was designed to address real needs of warehouse employees, making it a versatile tool for modern warehouses. The combination of voice and touch interfaces provides accessibility and efficiency improvements for warehouse workers.
